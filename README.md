# Pello Backend

This is the backend API for the Pello e-commerce project.

## Features

- User Authentication (Login/Register)
- Product Management
- Cart and Checkout
- Order Storage (MongoDB)
- Admin Panel to manage products and orders

## Tech Stack

- Node.js
- Express
- MongoDB (with Mongoose)
- JWT Auth
- Deployed on Render
